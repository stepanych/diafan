Минимальные технические требования при установке:
- mod_rewrite
- AllowOverride "All" для .htaccess
PHP 5.6+

Все технические требования https://www.diafan.ru/dokument/full-manual/introduction/tekhnicheskie_trebovaniya_k_khostingu/

Как установить DIAFAN.CMS https://www.diafan.ru/dokument/full-manual/introduction/install/

Если на Вашем хостинге установщик выдает ошибку 500 Internal Server Error, раскомментируйте строку 
RewriteBase /
в .htaccess.

Измените RewriteBase / на RewriteBase /diafan/ при установке DIAFAN.CMS в подпапку http://site.ru/diafan/

---------------------------------------------------------

ВНИМАНИЕ! Все файлы DIAFAN.CMS необходимо править редактором, поддерживаемым формат "UTF-8 without BOM". Встроенный в Windows стандартный "Блокнот" этот формат не поддерживает! Рекомендуется бесплатный редактор Notepad++ (http://notepad-plus-plus.org/download/v5.9.3.html)
